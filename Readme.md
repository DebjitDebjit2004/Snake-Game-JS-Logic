# 🐍 Snake Game - JS Logic Practice

A classic Snake Game built purely in free time to revise and practice Core JavaScript logics, DOM manipulation, and CSS Grid. 

## 🎮 Features
- **Classic Gameplay**: Eat food to grow your snake and increase your score.
- **Live Timer**: Track how long you survive in the game.
- **Dynamic Grid Setup**: The game board calculation handles grid blocks.
- **Custom Modals**: Clean UI for 'Start Game' and 'Game Over / Restart' screens.

## 🛠️ Technologies Used
- **HTML5**: Game structure and layout.
- **CSS3**: Styling, CSS Variables, Flexbox, and Grid layout.
- **Vanilla JavaScript**: Core game loop, array manipulations, collision detection, and event handling.

## 🚀 Purpose of the Project
This project was developed during free time as a fun exercise to brush up on:
- 2D grid generation and coordinate systems.
- Asynchronous execution and Game Loops using `setInterval`.
- Collision detection (walls and food).
- Keyboard event handling (`keydown`).

## 🏃 How to Play
1. Clone or download this repository.
2. Open `index.html` in your favorite web browser.
3. Click the **Start Game** button.
4. Use the **Arrow Keys** (Up, Down, Left, Right) to steer the snake.
5. Enjoy coding and gaming!
